## 📊 데이터 구조

### 주요 컬럼:

**기본 정보:**
- 사건번호: 경매 사건 번호
- 도로명주소, 지번주소: 물건 위치
- 용도: 아파트, 다세대, 단독주택 등

**금액 정보:**
- 감정가_num: 감정가 (숫자)
- 최저가_num: 최저 입찰가 (숫자)
- 낙찰가_num: 실제 낙찰가 (숫자)

**계산 변수:**
- 낙찰가율: 낙찰가 / 감정가 (타겟 변수)
- 최저가율: 최저가 / 감정가 (주요 피처)

**그룹:**
- 그룹: 0 (정상, ≥0.5) / 1 (유찰, <0.5)
- 년도: 2020~2025
